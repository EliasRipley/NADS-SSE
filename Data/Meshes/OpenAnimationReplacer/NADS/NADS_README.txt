NADS OAR Third-Person Scope

NADS ships template submods for animator integration, but no bundled dodge HKX clips.

Current runtime intent:
- Third-person dodge displacement is controller-driven in native code.
- Third-person visuals can use optional invisibility FX from the core mod INI.
- OAR support remains optional for user/authored clip packs.

Template folders:
- TP_Dodge_Frozen (neutral fallback stub)
- TP_Dodge_Forward / Right / Backward / Left
- TP_Dodge_RightForward / RightBackward / LeftBackward / LeftForward

For authoring details, see NADS_ANIMATOR_GUIDE.txt in this folder.
